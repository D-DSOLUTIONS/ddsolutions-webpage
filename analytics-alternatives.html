Alternativas de Analytics para DDSolutions

Opciones Gratuitas/Económicas:

1. Plausible Analytics (Recomendado)

✅ Sin cookies, cumple GDPR automáticamente

✅ Muy ligero (< 1KB)

💰 $9/mes o self-hosted gratis

2. Umami Analytics (Gratis)

✅ Open source y gratuito

✅ Puedes usar su servicio cloud o self-host

✅ Sin cookies, respeta privacidad

3. Simple Analytics

✅ Muy simple de usar

💰 $19/mes

4. GoatCounter (100% Gratis)

✅ Totalmente gratuito para uso personal/pequeño

✅ Open source

✅ Sin cookies

5. Splitbee (Ahora parte de Vercel)

✅ Gratis hasta 2,500 eventos/mes

✅ Integrado con Vercel

Para usar con GitHub Pages:

  1. Google Analytics - El más completo, gratis
  2. GoatCounter - 100% gratis, respetuoso con privacidad
  3. Umami Cloud - Gratis hasta 10k vistas/mes
  4. Plausible - $9/mes pero muy profesional

Configuración de Google Analytics:

  1. Ve a analytics.google.com
  2. Crea una cuenta y propiedad
  3. Selecciona "Web" como plataforma
  4. Ingresa tu URL de GitHub Pages
  5. Obtendrás un ID como "G-XXXXXXXXXX"
  6. Reemplaza el ID en el código que agregué

Datos que puedes ver:

  • 📊 Número de visitantes únicos
  • 🌍 País/ciudad de origen
  • 📱 Dispositivo (móvil/escritorio)
  • 🔗 Páginas más visitadas
  • ⏱️ Tiempo en el sitio
  • 🔄 Tasa de rebote
  • 📈 Fuentes de tráfico
  • 🔍 Términos de búsqueda
