## Quick Start

### Using Python (Recommended)
```bash
# Default port 8080
python3 server.py

# Custom port
python3 server.py 8888
```

### Using Node.js
```bash
npx http-server -p 8080
```

### Using PHP
```bash
php -S localhost:8080
```
